# Add Logo Here

## To complete the logo setup:

1. **Save the circular Mittag-Leffler silhouette logo** (the one with navy blue profile on cream circle)

2. **Name it**: `mittag-leffler-logo.png`

3. **Place it in**: `public/images/` folder (this directory)

4. **Then run**:
   ```bash
   git add public/images/mittag-leffler-logo.png
   git commit -m "Add official Mittag-Leffler logo"
   git push
   ```

The logo will automatically appear in:
- All page headers
- Browser favicon
- Railway app icon
